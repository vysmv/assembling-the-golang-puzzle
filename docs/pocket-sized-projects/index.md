## Проект Hello, earth!

ТЗ
Освоить темы:

- вывод данных в стандартный вывод (*standard output*);
- тестирование вывода в стандартный вывод;
- написание *table-driven* тестов;
- использование хеш-таблицы для хранения пар «ключ–значение»;
- использование флагов для чтения параметров командной строки.

Для этого необходимо:

- написать программу, которая принимает выбранный пользователем человеческий язык и выводит соответствующее приветствие;
- убедиться, что программа покрыта модульными тестами.

**Шаги**:

1. Создаем модуль:
```bash
go mod init github.com/vysmmv/hello-earth
```

2. Созаем main.go в корне проекта
```go
package main

import "fmt"

func main() {
	fmt.Println("Hello world")
}
```

3. Тестируем запуск:
```bash
go run main.go
```
